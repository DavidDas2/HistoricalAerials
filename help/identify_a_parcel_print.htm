Identify a Parcel

 
With the Parcel Layer turned on, click on any parcel.
 
A pop-up window will appear showing the Parcel Attributes.
 
 
Scroll down to view the Web Links associated with this Parcel.
 
View the Assessor Web Link
 
 
This opens the Maricopa County Assessor’s Page.
 
https://mcassessor.maricopa.gov/mcs/?q=21167062E
 
 
View the Treasurer Web Link
 
 
This opens the Maricopa County Treasurer’s Page.
 
https://treasurer.maricopa.gov/parcel/Summary.aspx
 
 
 
Notice that the pop-up window is partially blocking the view of the Parcel.
 
Click on the Dock Icon.
 
 
This moves the Pop-up window to the corner.
 
 
You have the option to undock.
 
Close the Pop-up
