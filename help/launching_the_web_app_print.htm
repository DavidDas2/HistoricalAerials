Launching the Web App

 
This is a browser-based Web Application. It works best with Google Chrome or Mozilla Firefox. They should be your first choice.
 
You may also use Internet Explorer 11, Microsoft Edge, or Safari 3.
 
It needs a minimum window size of 1100 pixels wide by 600 pixels high.
 
On tablet devices, you can use Safari for iOS or Chrome for Android.
Use in landscape mode only.
 
Depending on the size of your window, this is how your Web App may appear on first launch.
 
 
You can click on the link to the User Guide to launch the comprehensive HTML 5 Help System in a separate window.
 
Click OK to close the Splash Screen.
 
This is your initial view.
